### 1. Course Description

This course is designed to provide a comprehensive introduction to Artificial Intelligence (AI), covering fundamental concepts, algorithms, and applications. Students will explore problem-solving techniques, knowledge representation, deep learning and natural language processing (NLP). The course emphasizes both theoretical foundations and hands-on implementation using modern AI tools and frameworks (Python, TensorFlow, PyTorch, Scikit-Learn).

### 2. General Objectives

The course is designed with the following general objectives:

- To provide the students with the in-depth knowledge of intelligent agents, search algorithms, and knowledge representation techniques.
- To acquaint the students with the skill to design and implement machine learning and deep learning models.
- To equip the students with knowledge to explore real-world applications of AI in areas like reasoning and natural language processing.

### 3. Methods of Instruction

Lecture, Discussion, Readings, Case Studies, Project and Research Works.

### 5. Evaluation System and Students' Responsibilities

#### Evaluation System

The internal evaluation of a student may consist of assignments, attendance, internal assessment, project and research works etc. The internal evaluation scheme for this course is as follows (but not limited to):

| Internal Evaluation              | Weight | Marks  | External Evaluation      | Marks  |
| -------------------------------- | ------ | ------ | ------------------------ | ------ |
| Attendance & Class Participation | 10%    | 60     | Semester-End examination | 40     |
| Assignments                      | 20%    |        |                          |        |
| Internal Assessment              | 40%    |        |                          |        |
| Research/Project/Case Study Work | 30%    |        |                          |        |
| **Total Internal**               |        | **60** | **Total External**       | **40** |
| **Full Marks: 60 + 40 = 100**    |        |        |                          |        |

#### Students' Responsibilities

Each student must secure at least 60% marks separately in internal evaluation with 80% attendance in the class in order to appear in the Semester End Examination. Failing to get such a score will be given NOT QUALIFIED (NQ) to appear for the Semester-End Examinations. Students are advised to attend all the classes, formal exam, test, etc. and complete all the assignments within the specified time period. Students are required to complete all the requirements defined for the completion of the course.
